# xls2json
read xls data and transform to json (use SpreadsheetReader)


## demo
### map mode
* code
```
$ret = Xls2Json::process(__DIR__ . "/test.xls", 1);
echo json_encode($ret, JSON_PRETTY_PRINT);
```
* output
```
{
    "aaa": {
        "name": [
            "fly",
            "lily",
            "tom"
        ],
        "city": [
            "bj",
            "tj",
            "nj"
        ],
        "size": [
            10,
            20,
            ""
        ]
    },
    "bbb": {
        "game": [
            "a123",
            "z123"
        ],
        "money": [
            10,
            8
        ]
    }
}
```
